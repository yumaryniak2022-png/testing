## 1. Змінні та типи даних
Python підтримує різні типи даних: числа, рядки, списки, словники, множини, кортежі.

```python
# Приклад змінних
text_var = "Hello, Python!"
int_var = 10
float_var = 3.14
list_var = [1, "two", 3.0]
dict_var = {"key1": "value1", "key2": 2}
tuple_var = (1, 2, 3)
set_var = {1, 2, 3}
```
